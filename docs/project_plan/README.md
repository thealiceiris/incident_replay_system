# Incident Replay System Project Plan

This folder is the planning hub for finishing the Incident Replay System capstone in a 3 week window.

| File | Use it for |
| --- | --- |
| <a href="incident-replay-3-week-plan.md">3 Week Project Plan</a> | Scope, architecture, acceptance criteria, week goals, risks, and priorities. |
| <a href="actionable-task-lists.md">Small Actionable Task Lists</a> | Session-sized checklists for day-to-day execution. |

## Recommended Flow

| Step | Action |
| --- | --- |
| 1 | Read the 3 week plan once at the start of each week. |
| 2 | Pick one session from the task lists for each work block. |
| 3 | Keep the README, ADRs, and demo notes updated as implementation changes. |
| 4 | Use the final week for cleanup, evidence, and a reliable demo. |
